## 1. 游戏概述

### 一句话描述
> 在一个诅咒的中世纪轮回之境中，你是一位信仰者兼村长——每一轮投下骰子，在环形棋盘上闯关，带着战利品回乡建设村庄，让信仰在大地与高塔间流转。

### 游戏简介
《信仰轮回》是一款**单人回合制像素风轻肉鸽游戏**，使用 **Godot 4.x** 引擎开发。玩家扮演一位兼任村长的信仰者，在**环形棋盘**上通过**投掷骰子**移动，触发随机格子事件，以战斗、祈祷、交易与抉择逐层爬上「神之塔」。

游戏拥有**双主线结构**：
- **爬塔线**：在塔中探索，通过**装备驱动的 Build 系统**（无固定职业）自由定制角色风格
- **养成线**：将塔中获取的资源与信仰值贡献给村庄，发展村落，获得爬塔增益

**核心关键词**：骰子驱动 · 环形棋盘 · 装备定义角色 · 负重管理 · 信仰双线 · 村庄养成 · 像素美学

## 2. 核心体验目标

| 目标 | 说明 |
|------|------|
| **每局不同** | 棋盘格子随机生成，每次爬塔路径都是新体验 |
| **有意义的选择** | 骰子点数固定，但玩家选择方向、装备搭配、何时卸甲减重 |
| **装备塑造角色** | 没有固定职业，装备套装决定你的玩法风格和专属技能 |
| **负重是策略** | 带多少装备进塔、是否占用次元空间，都是关键决策 |
| **信仰双线流动** | 信仰值既驱动塔内能力，也在村庄中发挥建设作用 |
| **村庄有人情味** | 养成副玩法提供长期目标，与爬塔线形成正向循环 |
| **叙事氛围感** | 每个格子事件都有短暂文字叙事，构建中世纪世界感 |

## 3. 世界观与美术风格

### 3.1 世界观

时代背景是一个**架空中世纪欧洲**，正统教会正在衰落，神迹被质疑，异端横行。  
玩家踏入的「神之塔」是一座古老的试炼之地：  
- 传说中完成试炼者可获得「真理之言」——神明真正的意志  
- 每上升一层，世界观念越模糊——究竟何为信仰？何为盲从？  
- 故事无固定结局，结局由玩家的信仰值与选择决定

**主题对立**：  
- 秩序 vs 混沌  
- 圣洁 vs 亵渎  
- 服从 vs 自由意志  

### 3.2 美术风格

- **分辨率**：320×180 基础分辨率（像素放大 ×3~4 显示）  
- **调色盘**：暖黄/羊皮纸底色 + 深蓝/紫色阴影，点缀圣光金  
- **参考风格**：Unicorn Overlord 地图格 + Darkest Dungeon 暗调 + Moonlighter 明亮人物像素  
- **角色设计**：16×16 像素小人，战斗时 32×32 放大动画  
- **棋盘**：俯视角石板地图，骨骼纹理，有蜡烛、苔藓等装饰细节  
- **UI 框架**：羊皮纸卷轴风格，哥特字体，金属装饰边框

## 4. 技术基础

### 4.1 引擎选型：Godot 4.x

| 方面 | 说明 |
|------|------|
| **引擎版本** | Godot 4.x（推荐 4.3 或最新稳定版）|
| **渲染管线** | 2D 像素渲染，关闭抗锯齿，使用 `CanvasItem` 着色器实现像素发光/滤镜 |
| **场景结构** | 主场景 → 棋盘场景 / 战斗场景 / 村庄场景（通过 SceneTree 切换）|
| **数据管理** | 装备/遗物/卡牌数据使用 `Resource`（`.tres`）定义，便于编辑器配置 |
| **存档系统** | `FileAccess` + JSON 序列化，支持单槽自动存档 |
| **信号系统** | 全局 `EventBus`（AutoLoad 单例），解耦各模块通信 |
| **随机系统** | `RandomNumberGenerator` 配合种子，支持 Run 种子分享功能 |

### 4.2 项目结构建议

```
res://
├── scenes/
│   ├── board/        # 棋盘场景
│   ├── combat/       # 战斗场景
│   ├── village/      # 村庄养成场景
│   ├── ui/           # HUD、菜单、卡牌UI
│   └── world/        # 主世界/入口场景
├── scripts/
│   ├── systems/      # 核心系统（骰子/信仰/负重/Build）
│   ├── entities/     # 玩家、敌人、NPC
│   └── autoloads/    # EventBus, GameState, SaveManager
├── resources/
│   ├── equipment/    # 装备数据 .tres
│   ├── cards/        # 卡牌数据 .tres
│   ├── relics/       # 遗物数据 .tres
│   └── enemies/      # 敌人数据 .tres
├── assets/
│   ├── sprites/      # 像素图资源
│   ├── audio/        # 音效/BGM
│   └── fonts/        # 哥特像素字体
└── data/
    └── events/       # 随机事件 JSON 数据
```

### 4.3 关键 Godot 特性应用

- **TileMap**：村庄建设地图使用 TileMap 管理建筑摆放
- **AnimationPlayer**：骰子投掷动画、棋子移动动画
- **ShaderMaterial**：装备发光特效（骑士套装金光、异端套装紫焰）
- **SubViewport**：战斗场景嵌入棋盘UI，避免全场景切换
- **GDScript**：主要脚本语言，关键数值计算模块可用 C# 提速

## 5. 核心机制

### 4.1 游戏流程（每层）

```
┌─────────────────────────────────────────────────────────┐
│                      一层流程                            │
│                                                         │
│  开始新层 → 环形棋盘生成（8~12格）                        │
│      ↓                                                  │
│  投掷骰子（1~6点）→ 选择方向（顺/逆时针）→ 移动           │
│      ↓                                                  │
│  触发格子事件（战斗/商店/祈祷所/陷阱/宝藏/事件/BOSS）    │
│      ↓                                                  │
│  事件结算 → 回到棋盘中心（触发"层间结算"）               │
│      ↓                                                  │
│  选择：继续上升 or 在此层再转一圈（获取更多资源）         │
│      ↓                                                  │
│  上升到下一层（棋盘重新生成，格子数增加）                 │
└─────────────────────────────────────────────────────────┘
```

### 4.2 行动基本规则

- 每回合**强制投掷骰子**，点数即移动步数（不可拆分）
- 玩家选择**顺时针或逆时针**移动
- 落点格子**必须触发**，无法跳过
- 某些遗物或卡牌可以**修改骰子结果**（+1/-1/重投等）
- 棋盘可以**反复走圈**，但每多走一圈，"诅咒槽"上升

## 6. 骰子系统

### 5.1 基础骰子

| 骰子类型 | 获得方式 | 效果 |
|---------|---------|------|
| **普通骰（D6）** | 初始持有 | 投出 1~6 |
| **祝福骰（D4+2）** | 圣殿购买 | 投出 3~6，偏向高点 |
| **诅咒骰（2D3）** | 被诅咒后替换 | 投出 2~6，更多极端值 |
| **命运骰（D6×2 取高）** | 精英战掉落 | 两颗D6取高值 |
| **信仰骰** | 信仰值≥8时可选择使用 | 手动指定点数（消耗4信仰）|

### 5.2 骰子修正机制

玩家可以通过遗物、卡牌技能对骰子结果进行**修正**：

- **神迹牌**：将骰子结果 +1（每局可用1次）
- **预言遗物**：投骰前可看下一格是什么
- **祈祷师职业被动**：每次投出1时，获得2信仰值

## 7. 环形棋盘系统

### 6.1 棋盘结构

```
            [宝藏]
    [怪物]           [商店]
 
[祈祷]   ★中心★   [事件]
 
    [精英]           [陷阱]
            [BOSS出口]
```

- 棋盘为**圆环形**，格子数量：
  - 第1~3层：8格  
  - 第4~6层：10格  
  - 第7~9层：12格  
  - 第10层（最终层）：6格（纯BOSS格）

- 中心区域为"**神坛**"：  
  - 玩家每圈结束回到中心做层间抉择  
  - 神坛可激活特殊增益（消耗信仰值）

### 6.2 棋盘刷新规则

| 条件 | 刷新内容 |
|------|---------|
| 上升到新层 | 全部格子重新随机生成 |
| 再走一圈（诅咒圈） | 已触发格子变为"虚空格"（跳过，但+1诅咒） |
| 特殊事件触发 | 某格可能被"圣化"或"污染"，改变周围格子属性 |

## 8. 格子类型

### 7.1 基础格子

| 图标 | 名称 | 描述 | 频率 |
|------|------|------|------|
| ⚔️ | **战斗** | 随机普通怪物，战斗胜利获得金币+卡牌 | 高 |
| 💀 | **精英战** | 强力精英怪，必有稀有奖励 | 中 |
| 🏪 | **商店** | 购买卡牌/遗物/消耗品，用金币或信仰值 | 中 |
| 🕯️ | **祈祷所** | 获得信仰值，或触发神明特殊效果 | 中 |
| 📦 | **宝藏** | 随机获得卡牌/遗物/金币 | 低 |
| ❓ | **随机事件** | 文字选择事件，结果影响 Build | 中 |
| 🔥 | **陷阱** | 受到伤害或负面状态，但有反制奖励 | 中 |
| 🗡️ | **BOSS入口** | 每层至少一个，触发层BOSS战 | 固定 |
| ✨ | **圣化格** | 特殊强力格子，由事件触发 | 罕见 |
| 💀 | **虚空格** | 第二圈后空格，踩上+1诅咒 | 第二圈 |

### 7.2 随机事件示例

| 事件名 | 选项A | 选项B |
|--------|-------|-------|
| 十字路口的流浪汉 | 给他面包（-1金币，+2信仰） | 无视他（+0，-1信仰） |
| 异端的小册子 | 焚毁（+3信仰，获得"正统"标签） | 阅读（-2信仰，获得异端卡） |
| 倒塌的神坛 | 修复（消耗20金，+4信仰） | 取走遗物（获得遗物，-3信仰） |
| 临死的骑士 | 救治（付出5HP，获得传承卡） | 问询（获得情报，下层棋盘可预览） |
| 神明的低语 | 顺从（随机效果） | 质疑（-2信仰，但获得一张强力异端牌）|

## 9. 战斗系统

### 9.1 战斗基础

战斗基于**回合制卡牌**系统（类杀戮尖塔）：

- 每回合开始：抽3~5张手牌（受装备属性影响）
- 每回合有**行动点数（AP）**：初始为3点（力量属性可小幅影响）
- 打出卡牌消耗 AP；回合结束未打出的牌弃掉
- 敌人有**意图系统**：每回合显示下一步行动（攻击/格挡/召唤/诅咒）

### 9.2 战斗特色：骰子联动

部分卡牌效果与**本局骰子点数**挂钩：

- 「神判之刃」：造成 [骰子点数×2] 点伤害  
- 「幸运祈祷」：若本层骰子点数和 > 15，回复5HP  
- 「命运翻转」：将本回合已用骰子点数转化为护盾值  

### 9.3 状态效果

| 状态 | 效果 |
|------|------|
| 圣洁 | 每回合恢复2HP，可叠加 |
| 祝福 | 下一张卡费用-1 |
| 诅咒印记 | 每回合受到诅咒层数×1点伤害 |
| 异端之火 | 普通攻击额外附加灼烧 |
| 信念崩塌 | 信仰值归零，无法使用信仰技能3回合 |
| 神怒 | 攻击力×2，但每回合-10HP（过度使用神力的惩罚）|

## 10. 装备与 Build 系统

> **核心设计理念**：游戏没有固定职业。玩家通过收集和穿戴装备，自由组合出战斗风格。装备套装触发专属技能，形成类职业的体验，但完全由玩家自主选择。

### 10.1 属性系统

玩家有以下核心属性，均可通过装备、遗物、升级提升：

| 属性 | 说明 |
|------|------|
| **力量（STR）** | 增加物理攻击伤害；**提升负重上限**（每点+2kg） |
| **敏捷（AGI）** | 增加闪避率；战斗时先手概率；棋盘移动某些格子触发概率 |
| **智力（INT）** | 增加法术/信仰技能伤害；增加卡牌效果数值 |
| **体质（CON）** | 增加最大HP；提升护甲值上限 |
| **信念（FAI）** | 提高信仰值上限；信仰获取效率+（与信仰系统深度联动）|

### 10.2 装备槽位

```
┌────────────────────────────────────────┐
│  头盔   │  胸甲   │  护腕   │  腿甲    │
│  武器   │  副手   │  戒指×2 │  项链    │
│  披风   │  腰带   │  靴子   │          │
└────────────────────────────────────────┘
```

- 共 **11个槽位**（戒指占2格）
- 每件装备有**重量值（kg）**
- 玩家**负重上限 = 基础10kg + 力量×2kg**

### 10.3 装备套装与专属技能

穿戴同一套装的**足够件数**后，自动解锁该套装的专属被动/主动技能：

#### 🛡️ 骑士套装（Knight's Set）
> 沉重铁甲，神圣加持，庄严而不可撼动

| 件数 | 套装效果 |
|------|---------|
| 2件 | 每回合自动获得4点护甲 |
| 4件 | 解锁主动技能「神圣冲锋」：消耗5信仰，对所有敌人造成(力量×3)伤害 |
| 6件（满套）| 受到致命伤害时1次免死，保留1HP（每场战斗1次）|

**重量**：总重约 **22kg**，需要高力量才能全套穿戴

#### 📿 修士袍装（Friar's Robe）
> 轻薄布衣，信仰刺绣，每一针都是祈祷

| 件数 | 套装效果 |
|------|---------|
| 2件 | 信仰值每满5点，自动回复3HP |
| 4件 | 解锁主动技能「神谕降临」：消耗8信仰，随机施加3个祝福状态并抽2张牌 |
| 6件（满套）| 每次获得信仰值时，额外获得1点（永久被动）|

**重量**：总重约 **6kg**，轻装几乎无负担

#### 🔥 异端装束（Heretic's Garb）
> 破碎的符文皮甲，禁忌的紫色火焰渗透每一条缝隙

| 件数 | 套装效果 |
|------|---------|
| 2件 | 信仰值越低，法术伤害越高（最多+80%） |
| 4件 | 解锁主动技能「禁断仪式」：消耗3信仰（或等量HP），召唤异端火焰持续灼烧敌人 |
| 6件（满套）| 每次打出异端牌，额外造成2点穿透伤害 |

**重量**：总重约 **10kg**，中等负重

#### 🗡️ 暗影盗贼套（Shadow Thief Set）
> 不存在于教会记录中的装备，来自某个更古老的秘密

| 件数 | 套装效果 |
|------|---------|
| 2件 | 闪避率+15% |
| 4件 | 解锁主动技能「消影步」：本回合所有攻击无法被格挡，消耗3AP |
| 6件（满套）| 战斗开始时随机窃取敌人1个正面状态 |

**重量**：总重约 **8kg**，轻便

#### ⚗️ 炼金术士套（Alchemist Set）
> 奇异的药瓶、符文手套、蒸馏护目镜——实验室的延伸

| 件数 | 套装效果 |
|------|---------|
| 2件 | 每层开始随机获得1个消耗品 |
| 4件 | 解锁主动技能「实验爆炸」：将手中1张消耗品转化为对所有敌人的伤害 |
| 6件（满套）| 消耗品效果翻倍 |

**重量**：总重约 **12kg**，中等

### 10.4 混搭机制

玩家可以**自由混搭**不同套装，例如：
- 骑士胸甲（高护甲）+ 修士袍头巾（信仰加成）+ 异端皮手套（法术增伤）
- 无法触发任何满套效果，但属性灵活叠加
- **混搭鼓励策略**：某些遗物专门奖励混搭风格（如「百搭徽章」：未触发任何套装时，所有属性+2）

## 11. 负重与背包系统

> 重量是战略资源，不只是限制。

### 11.1 负重机制

```
负重上限 = 10kg（基础）+ 力量属性 × 2kg
当前负重 = 所有已装备物品重量之和 + 背包中携带物品重量
```

**负重状态**：

| 状态 | 条件 | 效果 |
|------|------|------|
| ✅ 轻装 | 负重 ≤ 上限×50% | 闪避率+10%，骰子结果+1（每层1次）|
| ⚖️ 正常 | 50%~80% | 无额外效果 |
| ⚠️ 超重 | 80%~100% | 移速动画变慢（演出效果），战斗先手概率-20% |
| ❌ 严重超重 | >100% | 每踩1格受到2点疲劳伤害，战斗AP-1 |

### 11.2 背包系统

- 玩家有一个**普通背包**：初始容量 **6格**，每格1kg负重计入
- 背包中的物品（装备/遗物/消耗品）**不在身上时不触发被动效果**
- 可在塔中商店购买背包扩容（+2格，有上限）

### 11.3 次元空间（Dimensional Pouch）

**次元空间**是一种特殊的空间扩展法术/遗物，提供**不计入负重的额外储存格**：

| 名称 | 获得方式 | 容量 | 特性 |
|------|---------|------|------|
| 次元小袋 | 商店购买/宝藏获得 | +4格，0重量 | 基础次元容器 |
| 异端法阵 | 异端格子事件 | +8格，0重量 | 代价：进入时-2信仰 |
| 神圣容器 | 圣化格奖励 | +6格，0重量 | 存放的装备获得+1圣洁层数 |
| 虚空裂缝 | BOSS掉落（稀有）| +16格，0重量 | 随机：取出物品时有5%概率消失 |

**次元空间限制**：
- 次元空间中的物品**不影响负重**，但**取用有成本**（1AP 或特定条件）
- 某些格子事件可能导致次元空间**不稳定**，暂时无法访问
- 村庄升级后可在「仓库」永久存放部分物品（下一次 Run 可携带出发）

### 11.4 管理策略示例

```
场景：玩家背包满了，发现了一件强力的骑士胸甲
选项A：装备它，卸下现有胸甲丢弃
选项B：放入次元空间（消耗1AP获取）
选项C：放入背包，但超重了——接下来几格会受到疲劳伤害
选项D：回村时存入仓库（需先结束本层）
```

## 12. 信仰系统

> 信仰是本游戏最核心的差异化机制——它在塔内是战斗资源，在村庄中是建设货币，两条线共享同一个"信仰池"，形成取舍张力。

### 12.1 信仰值（Faith Points）

- **塔内信仰**：Run 内使用，初始 **5点**（满值20点，可通过村庄升级提高上限）
- **村庄信仰**：每次 Run 结算后，玩家选择将部分塔内信仰贡献给村庄（不可撤回）
- **二者权衡**：留在塔内的信仰提升当局胜率；贡献给村庄的信仰推动长期养成

### 12.2 塔内获得与消耗

**获得方式**：
- 踩到祈祷所 +2~4
- 做出"信仰选项"的事件选择 +1~3
- 击败亵渎型怪物 +1
- 轻装状态下踩到圣化格 +3
- 某些装备/遗物被动获取

**消耗方式**：
- 使用信仰骰（指定点数）：-4
- 激活套装专属神力技能：-3~8
- 在商店购买圣物：-3~8
- 选择"亵渎选项"：-2~4

### 12.3 信仰标签系统

玩家积累信仰选择，会获得**标签**，影响全局效果：

| 标签 | 获得条件 | 效果 |
|------|---------|------|
| **圣战者** | 信仰值5次达到最大值 | 攻击亵渎型敌人+50%伤害 |
| **异端者** | 3次选择亵渎选项 | 解锁异端卡牌池，信仰上限-5 |
| **殉道者** | HP低于20%时触发战斗 | 低血量时攻击力+2，但不能回血 |
| **怀疑者** | 5次拒绝神明指引 | 所有随机事件结果可重投一次 |
| **虔信者** | 全程信仰值不低于10 | 每层开始额外抽一张圣洁卡 |
| **慷慨村长** | 累计贡献信仰值 ≥ 50 | 村庄加成反哺塔内：每层起始+1信仰 |

## 13. 村庄养成系统

> 玩家是村长，塔是副业，村庄是家。每次冒险都是为了让村子活下去，变得更好。

### 13.1 系统概览

村庄养成是**跨 Run 的永久进度系统**，与爬塔形成**资源循环**：

```
爬塔 → 获取资源（金币/材料/信仰） → 贡献给村庄
                                       ↓
                              村庄升级 → 解锁新建筑/NPC
                                       ↓
                              新建筑/NPC → 提供爬塔增益
                                       ↓
                              更强地爬塔 → 获取更多资源
```

### 13.2 村庄等级（定居点进化）

| 等级 | 名称 | 升级条件 | 解锁内容 |
|------|------|---------|---------|
| 1 | **村落** | 初始状态 | 基础设施（家、祈祷场、出发点）|
| 2 | **村庄** | 贡献资源×50 + 信仰×20 | 铁匠铺、市场、第1个建筑槽 |
| 3 | **镇子** | 贡献资源×200 + 信仰×80 | 城墙、教堂、招募高级村民 |
| 4 | **城市** | 贡献资源×600 + 信仰×200 | 大学、骑士团总部、商会 |
| 5 | **圣城** | 特殊条件（完成真结局分支）| 神殿、永久爬塔增益最大化 |

### 13.3 信仰值在村庄中的功能

信仰值是村庄的**第二货币**，专用于以下功能：

| 功能 | 消耗信仰 | 效果 |
|------|---------|------|
| **招募村民** | 5~20信仰/人 | 村民执行各类任务，人口增加税收 |
| **升级定居点** | 参见上表 | 推进村庄等级 |
| **建造圣所** | 30信仰 | 解锁特殊祈祷功能，Run开始时额外+2信仰 |
| **神明祭典** | 50信仰 | 全村民效率+50%（持续3次Run）|
| **赦免罪人** | 10信仰 | 将因事件失去的村民找回 |
| **指派传教士** | 15信仰 | 派村民去邻村，带回稀有材料或新村民 |

### 13.4 税收系统

- **人口**决定税收基础：每个村民每次 Run 结束贡献 1~3 金币（取决于职业）
- **信仰值**影响税收效率：信仰高 → 村民劳动意愿强 → 税收×1.2
- **建筑**提供税收加成（如市场+20%，城墙+5%安全保障）
- 玩家 Run 开始前，从村庄领取**出发补给**（金币+可选消耗品）

### 13.5 建筑系统

| 建筑 | 解锁等级 | 建造成本 | 功能 |
|------|---------|---------|------|
| **铁匠铺** | 村庄 | 资源×30+金×50 | 每次 Run 出发时可强化1件已有装备 |
| **市场** | 村庄 | 资源×20+金×30 | 商店内货物种类+2，价格-10% |
| **教堂** | 镇子 | 资源×80+信仰×30 | Run开始信仰上限+3；祈祷所效果×1.5 |
| **图书馆** | 镇子 | 资源×60+金×100 | 每次 Run 开始时可预览1个随机事件结果 |
| **次元研究所** | 城市 | 资源×150+信仰×50 | 次元空间容量+4；取用物品不消耗AP |
| **仓库** | 村庄 | 资源×40 | 可跨 Run 存放最多4件装备/遗物 |
| **骑士团总部** | 城市 | 资源×200+信仰×60 | 解锁稀有套装碎片掉落；Run开始可携带1件稀有装备 |

### 13.6 村民系统

村民有**职业**，执行不同任务，提供不同加成：

| 村民职业 | 招募信仰 | 专长 |
|---------|---------|------|
| 农民 | 5信仰 | 基础税收来源，人口核心 |
| 铁匠 | 10信仰 | 强化装备效果+1 |
| 牧师 | 15信仰 | 每次Run结算后恢复部分信仰值 |
| 商人 | 12信仰 | 商店出现稀有货物概率+15% |
| 传教士 | 15信仰 | 可派出探索，带回资源/情报 |
| 骑士 | 20信仰 | 可作为傀儡参与1次战斗（每局1次）|

### 13.7 村庄危机事件

村庄并非安全港——随机事件可能影响村庄状态：

| 事件 | 触发条件 | 效果 | 应对 |
|------|---------|------|------|
| 瘟疫 | 随机（低概率）| 人口-2，税收-30% | 消耗医疗资源解决 |
| 异端暴动 | 信仰值长期低 | 村民逃跑-3人 | 举行祭典稳定民心 |
| 强盗袭击 | 无城墙且人口高 | 资源仓库被抢 | 骑士护卫或高城墙防御 |
| 大丰收 | 随机（低概率）| 税收×2（持续1次Run）| 无需应对，正面事件 |

## 16. BOSS 设计

### 16.1 BOSS 机制通则

- 每层一个 BOSS，有专属像素立绘和战斗主题曲
- BOSS 有**阶段变化**：通常2个阶段
- BOSS 会根据**玩家装备套装**和**信仰值**调整策略：
  - 骑士套装满激活 → BOSS 强化对护甲的穿透
  - 高信仰 → BOSS 使用"亵渎攻击"克制圣洁
  - 低信仰 → BOSS 使用"秩序惩罚"攻击异端

### 16.2 BOSS 列表（概念）

| 层 | BOSS名 | 主题 | 特殊机制 |
|----|--------|------|---------|
| 3  | 腐败审判官 | 堕落的秩序 | 每回合添加1张诅咒牌进手牌 |
| 6  | 圣母像·暗面 | 信仰的扭曲 | 模仿玩家上一张打出的卡牌 |
| 9  | 大祭司遗体 | 教会的牺牲 | 每受到一次攻击，恢复少量HP |
| 10 | 神明本体·「真理」 | 终极审判 | 根据玩家信仰标签+村庄等级改变形态和技能 |

## 17. 肉鸽要素

### 17.1 每局随机内容

- 棋盘格子排布完全随机
- 怪物组合随机（含词缀系统）
- 卡牌/装备奖励随机（受当前套装偏向）
- 遗物随机（共约60个，每局约见到15~20个）
- 事件随机（共约40个，每局约触发10~15个）
- 商店货架随机

### 17.2 永久解锁（Meta 进度）

| 解锁条件 | 解锁内容 |
|---------|---------|
| 首次通关 | 困难模式（棋盘格+诅咒加速）|
| 村庄升至镇子 | 解锁稀有装备池 |
| 获得全部信仰标签 | 真结局分支解锁 |
| 完成10次殉道者通关 | 特殊遗物「殉道者圣冠」永久解锁 |
| 村庄升至圣城 | 最终结局解锁，特殊开场叙事 |

### 17.3 诅咒等级（Run内难度调节）

| 诅咒层数 | 效果 |
|---------|------|
| 0-2 | 无附加效果 |
| 3-5 | 怪物+10% HP |
| 6-8 | 商店价格×1.5 |
| 9-11 | 每回合棋盘随机移动1格 |
| 12+ | 每步移动额外触发随机负面事件 |

## 19. 音效与音乐方向

### 19.1 音乐风格

- **主题**：中世纪圣歌 + 低保真电子节拍（Lo-fi Gregorian）
- **村庄 BGM**：温暖羽管键琴 + 自然音效（风声、鸟鸣），祥和感
- **棋盘探索 BGM**：轻缓管风琴 + 轻鼓点
- **战斗 BGM**：紧张弦乐 + 打击乐，精英战加入合唱片段
- **BOSS 战 BGM**：全力管风琴 + 黑暗合唱
- **神坛抉择 BGM**：空灵单音铃声，宁静而压迫
- **村庄危机 BGM**：不安的弦乐，低沉鼓点

### 19.2 音效设计

- 骰子投掷：骨头/石头碰撞声
- 棋子移动：石板摩擦声
- 格子激活：蜡烛点燃/门锁开启声
- 攻击音效：金属碰撞，法术有圣光/火焰质感
- 信仰获得：短促合唱音效（类似「Amen」片段）
- 获得诅咒：低沉嗡鸣声
- 套装激活：对应风格音效（骑士：号角；修士：钟声；异端：诡异低鸣）
- 负重超限：沉重喘息声
- 村庄升级：欢呼声 + 教堂钟声

## 附录 A：术语表

| 术语 | 说明 |
|------|------|
| Run | 一局完整游戏流程（从出发到死亡/通关）|
| Build | 本局装备套装+卡牌+遗物的搭配方案 |
| AP | 行动点数，每回合使用卡牌的资源 |
| 诅咒槽 | 绕圈惩罚机制，越走越危险 |
| 信仰值 | Faith Points，塔内战斗资源 & 村庄建设货币 |
| 层间神坛 | 每层结束后的增益选择界面 |
| 负重 | 装备和背包总重量，超限有惩罚 |
| 次元空间 | 不计入负重的特殊储存容器 |
| 村庄等级 | 跨Run的永久养成进度 |
