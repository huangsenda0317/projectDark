# 01 · 格子事件表 v0

## 1. 分类（用于数据表 `category` 字段）

| 代码 | 名称 | 说明 |
|------|------|------|
| `resource` | 资源与成长 | 金币、经验、属性选择、临时 Buff |
| `movement` | 移动与位置 | 再动、后退、传送、强制移位 |
| `floor` | 楼层与结构 | 上楼、下楼、隐藏楼梯、支路 |
| `harm` | 伤害与压力 | 直伤、资源侵蚀、暴露值 |
| `choice` | 选择与剧情 | 二选一 / 三选一风险回报 |
| `special` | 稀有机制 | 镜像、赌徒、地图改写、肃清计时 |

## 2. 标签（用于 `tag` 字段）

| 标签 | 含义 |
|------|------|
| `好` | 主反馈为收益或可控收益 |
| `坏` | 主反馈为损失或高压 |
| `中性` | 视构筑或选择而定 |
| `楼层改变` | 可能改变当前层/跨层状态 |

## 3. 权重说明

- **act1 / act2 / act3**：对应层段约 **1–33 / 34–66 / 67–99**，相对权重（同一层段内事件池归一化用）。
- v0 原则：act1 少即死、少强贬损；act3 多抉择、机制格略升。
