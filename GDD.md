## 1. 游戏概述

### 一句话描述

> 在一个诅咒的中世纪轮回之境中，你是一位信仰者兼村长——收集并驾驭骰子之灵，在环形棋盘上闯关，将骰子嵌入装备释放真正力量，带着战利品回乡建设村庄，让信仰在大地与高塔间流转。

### 游戏简介

《骰境：信仰轮回》是一款**单人回合制像素风轻肉鸽游戏**，使用 **Godot 4.6** 引擎开发。玩家扮演一位兼任村长的信仰者，在**环形棋盘**上通过**投掷骰子**移动，触发随机格子事件，以战斗、祈祷、交易与抉择逐层爬上「神之塔」。

骰子是本游戏的**唯一核心道具**——不再是一次性消耗品，而是可收集、有品质、带词条的实体宝物。骰子同时驱动三个系统：**移动（投掷步数）、战斗（嵌入装备释放骰子公式）、探索（触发特殊格子）**。

游戏拥有**双主线结构**：

- **爬塔线**：在塔中探索，收集稀有骰子嵌入装备，通过**骰子驱动的 Build 系统**（无固定职业）自由定制角色风格
- **养成线**：将塔中获取的资源与信仰值贡献给村庄，发展村落，获得爬塔增益

**核心关键词**：骰子收集 · 环形棋盘 · 骰子嵌入赋能 · 负重管理 · 信仰双线 · 村庄养成 · 像素美学 · 多主题换皮

## 2. 核心体验目标

| 目标             | 说明                                                 |
| ---------------- | ---------------------------------------------------- |
| **每局不同**     | 棋盘格子随机生成，每次爬塔路径都是新体验             |
| **有意义的选择** | 骰子点数固定，但玩家选择方向、装备搭配、何时卸甲减重 |
| **装备塑造角色** | 没有固定职业，装备套装决定你的玩法风格和专属技能     |
| **负重是策略**   | 带多少装备进塔、是否占用次元空间，都是关键决策       |
| **信仰双线流动** | 信仰值既驱动塔内能力，也在村庄中发挥建设作用         |
| **村庄有人情味** | 养成副玩法提供长期目标，与爬塔线形成正向循环         |
| **叙事氛围感**   | 每个格子事件都有短暂文字叙事，构建中世纪世界感       |

## 3. 世界观与美术风格

### 3.1 世界观

时代背景是一个**架空中世纪欧洲**，正统教会正在衰落，神迹被质疑，异端横行。  
玩家踏入的「神之塔」是一座古老的试炼之地：

- 传说中完成试炼者可获得「真理之言」——神明真正的意志
- 每上升一层，世界观念越模糊——究竟何为信仰？何为盲从？
- 故事无固定结局，结局由玩家的信仰值与选择决定

**主题对立**：

- 秩序 vs 混沌
- 圣洁 vs 亵渎
- 服从 vs 自由意志

### 3.2 美术风格

- **分辨率**：320×180 基础分辨率（像素放大 ×3~4 显示）
- **调色盘**：暖黄/羊皮纸底色 + 深蓝/紫色阴影，点缀圣光金
- **参考风格**：Unicorn Overlord 地图格 + Darkest Dungeon 暗调 + Moonlighter 明亮人物像素
- **角色设计**：16×16 像素小人，战斗时 32×32 放大动画
- **棋盘**：俯视角石板地图，骨骼纹理，有蜡烛、苔藓等装饰细节
- **UI 框架**：羊皮纸卷轴风格，哥特字体，金属装饰边框

### 3.3 多主题换皮系统

「骰境」系列支持主题切换，玩家在主界面选择主题即可整体换皮。核心玩法不变，视听层全面替换。

| 主题 | 代号 | 骰子材质 | 棋盘风格 | 敌人风格 | BGM 基调 |
|------|------|----------|----------|----------|----------|
| **中世纪地下城** | 信仰轮回 | 骨/石/金属 | 石板纹理 | 骑士/修士/异端 | 管风琴 + 圣歌 |
| **中国洪荒** | 洪荒纪 | 玉石/青铜/兽骨 | 太极云纹 | 妖兽/巫族/散仙 | 古筝 + 鼓乐 |
| **中国仙侠** | 剑仙问道 | 灵石/仙玉/剑魄 | 灵山仙雾 | 妖魔/剑修/邪道 | 笛箫 + 编钟 |
| **日本百鬼** | 百鬼夜行 | 勾玉/符纸/妖骨 | 神社鸟居 | 妖怪/阴阳师/鬼族 | 三味线 + 太鼓 |

**主题切换规则**：
- 主界面选择主题，切换后全 Run 生效
- 不同主题的骰子外观不同，但品质/面数/词条机制完全一致
- 敌人名称、事件文本、UI 边框同步切换
- 村庄建筑外观随主题变化
- 未来可扩展 Steam 创意工坊，开放玩家自制主题

## 4. 技术基础

### 4.1 引擎选型：Godot 4.x

| 方面         | 说明                                                               |
| ------------ | ------------------------------------------------------------------ |
| **引擎版本** | Godot 4.6（详见 docs/engine-reference/godot/VERSION.md）             |
| **渲染管线** | 2D 像素渲染，关闭抗锯齿，使用 `CanvasItem` 着色器实现像素发光/滤镜 |
| **场景结构** | 主场景 → 棋盘场景 / 战斗场景 / 村庄场景（通过 SceneTree 切换）     |
| **数据管理** | 装备/遗物/道具数据使用 `Resource`（`.tres`）定义，便于编辑器配置   |
| **存档系统** | `FileAccess` + JSON 序列化，支持单槽自动存档                       |
| **信号系统** | 全局 `EventBus`（AutoLoad 单例），解耦各模块通信                   |
| **随机系统** | `RandomNumberGenerator` 配合种子，支持 Run 种子分享功能            |

### 4.2 项目结构建议

```
res://
├── scenes/
│   ├── board/        # 棋盘场景
│   ├── combat/       # 战斗场景
│   ├── village/      # 村庄养成场景
│   ├── ui/           # HUD、菜单、奖励选择UI
│   └── world/        # 主世界/入口场景
├── scripts/
│   ├── systems/      # 核心系统（骰子/信仰/负重/Build）
│   ├── entities/     # 玩家、敌人、NPC
│   └── autoloads/    # EventBus, GameState, SaveManager
├── resources/
│   ├── equipment/    # 装备数据 .tres
│   ├── items/        # 道具数据 .tres
│   ├── relics/       # 遗物数据 .tres
│   └── enemies/      # 敌人数据 .tres
├── assets/
│   ├── sprites/      # 像素图资源
│   ├── audio/        # 音效/BGM
│   └── fonts/        # 哥特像素字体
└── data/
    └── events/       # 随机事件 JSON 数据
```

### 4.3 关键 Godot 特性应用

- **TileMap**：村庄建设地图使用 TileMap 管理建筑摆放
- **AnimationPlayer**：骰子投掷动画、棋子移动动画
- **ShaderMaterial**：装备发光特效（骑士套装金光、异端套装紫焰）
- **SubViewport**：战斗场景嵌入棋盘 UI，避免全场景切换
- **GDScript**：主要脚本语言，关键数值计算模块可用 C# 提速

### 5.1 游戏流程（宏观）

```
┌─────────────────────────────────────────────────────────┐
│                   一次 Run 的宏观流程                     │
│                                                         │
│  村庄大厅 → 进入地图室，查看世界地图                      │
│      ↓                                                  │
│  选择一座已解锁的塔 → 消耗旅途补给 → 出发                │
│      ↓                                                  │
│  进入塔 → 塔门准备（嵌入骰子、装备调整）                  │
│      ↓                                                  │
│  ┌── 塔内循环 ──────────────────────────┐               │
│  │                                       │               │
│  │  新层开始 → 环形棋盘生成               │               │
│  │      ↓                                │               │
│  │  选择骰子投掷 → 顺/逆时针移动           │               │
│  │      ↓                                │               │
│  │  触发格子事件（战斗/商店/祈祷所/...）   │               │
│  │      ↓                                │               │
│  │  层间结算（神坛 → 继续上升/绕圈刷）     │               │
│  │      ↓                                │               │
│  │  击败 BOSS → 塔通关                    │               │
│  └──────────────────────────────────────┘               │
│      ↓                                                  │
│  返回村庄 → 贡献资源 → 结算                              │
│      ↓                                                  │
│  选择：继续探索下一座塔 or 结束 Run                      │
└─────────────────────────────────────────────────────────┘
```

### 5.2 塔内一层流程

```
┌─────────────────────────────────────────────────────────┐
│                      一层流程                            │
│                                                         │
│  开始新层 → 环形棋盘生成（24~44格，随机）                 │
│      ↓                                                  │
│  选择骰子投掷 → 选择方向（顺/逆时针）→ 移动              │
│      ↓                                                  │
│  触发格子事件（战斗/商店/祈祷所/陷阱/宝藏/事件/BOSS）    │
│      ↓                                                  │
│  事件结算 → 回到棋盘中心（触发"层间结算"）               │
│      ↓                                                  │
│  选择：继续上升 or 在此层再转一圈（获取更多资源）         │
│      ↓                                                  │
│  上升到下一层（棋盘重新生成，格子数 +2）                  │
└─────────────────────────────────────────────────────────┘
```

### 5.3 行动基本规则

- 每回合从骰子匣**选择一枚骰子投掷**，点数即移动步数（投掷后骰子归还匣中）
- 玩家选择**顺时针或逆时针**移动
- 落点格子**必须触发**，无法跳过
- 某些遗物可以**修改骰子结果**（+1/-1/重投等）
- 环形棋盘可以**套圈**（高点数绕多圈），每多走一圈「诅咒槽」+1
- 进入塔内所有格子**初始不可见**，踩上才知道内容

## 6. 骰子系统（核心道具）

> **核心设计理念**：骰子不再是消耗性随机数生成器，而是可收集、有品质、带词条的实体宝物。它是本游戏的唯一核心道具，同时驱动**移动**、**战斗**、**探索**三大系统。玩家的一切决策都围绕骰子展开。

### 6.1 骰子概述

一枚骰子同时服务于三个维度：

```
            ┌──────────────┐
            │   骰子实体    │
            │ (品质+词条)   │
            │ (面数D4~D100) │
            └───┬───┬───┬──┘
           ┌────┘   │   └────┐
           ↓        ↓        ↓
      棋盘移动    装备嵌入   格子触发
     (投掷步数)  (释放公式) (特殊事件)
```

- **移动**：选一枚骰子投掷，点数 = 步数
- **嵌入**：将骰子嵌入装备骰槽，激活装备的骰子伤害公式和词条加成
- **触发**：某些特殊格子需要特定品质/面数的骰子来解锁

### 6.2 骰子品质与词条

| 品质 | 稀有度色 | 词条数 | 掉落层数 | 说明 |
|------|----------|--------|----------|------|
| **普通** | 白 | 1 条 | 1-3 层 | 常见，基础词条 |
| **精良** | 蓝 | 2 条 | 1-6 层 | 较常见，实用词条 |
| **稀有** | 紫 | 3 条 | 4-10 层 | 强力词条组合 |
| **传说** | 金 | 4 条 | 7-10 层 | 顶级词条，稀有 |

### 6.3 词条类型

词条按生效场景分为三类：

#### 投掷词条（移动时触发）

| 词条 | 效果 | 品质 |
|------|------|------|
| 步数修正 | 投掷后点数 ±1（可选） | 白+ |
| 先手之骰 | 踩战斗格时先手 +1 | 蓝+ |
| 商道 | 经过商店格时自动触发 9 折 | 蓝+ |
| 疾走 | 顺时针移动时额外 +1 步 | 紫+ |
| 预知 | 投掷前可预览落点格类型 | 紫+ |
| 轮回步 | 可逆时针走完一圈回到原地（不触发诅咒） | 金 |

#### 嵌入词条（嵌入装备时生效）

| 词条 | 效果 | 品质 |
|------|------|------|
| 锋锐 | 嵌入武器时暴击率 +5% | 白+ |
| 坚韧 | 嵌入防具时护甲 +2 | 白+ |
| 灼热 | 攻击附带骰子面数×0.5 的火焰伤害 | 蓝+ |
| 吸血 | 造成伤害的 10% 转为 HP | 紫+ |
| 碎裂 | 暴击时目标护甲减半 | 紫+ |
| 神罚 | 对亵渎敌人伤害 ×1.5 | 金 |

#### 通用词条（持有即生效，无需嵌入）

| 词条 | 效果 | 品质 |
|------|------|------|
| 治愈之光 | 每层开始恢复 3 HP | 白+ |
| 虔诚 | 信仰获取 +1 | 蓝+ |
| 轻负 | 负重上限 +2kg | 蓝+ |
| 幸运星 | 战斗奖励三选一出现稀有选项概率 +10% | 紫+ |
| 骰子磁铁 | 战斗掉落额外骰子的概率 +15% | 金 |

### 6.4 骰子面数

骰子面数决定了投掷范围和嵌入威力。面数越大越稀有。

| 面数 | 移动范围 | 嵌入伤害基础 | 掉落起始层 | 掉落率 | 备注 |
|------|----------|-------------|-----------|--------|------|
| **D4** | 1-4 步 | 1d4 | 1 层 | 极高 | 初始兜底骰子 |
| **D6** | 1-6 步 | 1d6 | 1 层 | 极高 | 初始赠送 |
| **D8** | 1-8 步 | 1d8 | 1 层 | 高 | 早期主力 |
| **D10** | 1-10 步 | 1d10 | 4 层 | 中 | 中期升级 |
| **D12** | 1-12 步 | 1d12 | 4 层 | 中 | 中期主力 |
| **D20** | 1-20 步（取半：1-10） | 1d20 | 7 层 | 低 | 传说级以下最强 |
| **D100** | ❌ 不可用于移动 | 1d100 | 全层 | 极低 | 传说骰子，专属传奇词条池 |

**面数移动规则**：
- D20 移动时取半（向下取整，1-10），防止一圈走完毫无探索感
- D100 不可用于移动——太重了，掷不动。仅供嵌入战斗使用
- 环形棋盘可套圈，高点数 = 快速绕圈、多次触发

**D100 传奇词条池（专属）**：

| 词条 | 效果 |
|------|------|
| 神明意志 | 投出 100 即死任意敌人（BOSS 减半血） |
| 创世之骰 | 嵌入后全属性 +5 |
| 命运重铸 | 每场战斗可重投 1 次伤害（取高） |
| 百面归一 | 嵌入后武器伤害公式升级为 2d50 |

### 6.5 骰子匣

骰子匣是玩家的**骰子携带容器**，初始容量极小，扩容是核心成长线。

| 属性 | 初始值 | 上限 |
|------|--------|------|
| 骰子匣格子 | **2 格** | 10 格 |
| 初始骰子 | **1 枚 D6（白）** | — |

**扩容方式**：

| 方式 | 条件 | 说明 |
|------|------|------|
| BOSS 掉落 | 每层 BOSS 击杀后，三选一奖励中**低概率**出现 +1 格 | 主要来源 |
| 商店购买 | 高价（50 金币 / 格，每 Run 限购 1 次） | 补充来源 |
| 铁匠铺升级 | 村庄铁匠铺 Lv2+ 可永久 +1 格（跨 Run） | 永久进度 |

**兜底机制**：
- 骰子匣为空时，出发前可向村庄 NPC **免费领取 1 枚 D4 白色骰子**
- 塔内商店**固定出售 D4 骰子**（10 金币），确保不会软锁

### 6.6 骰子用途一：移动（棋盘骰子）

每回合选择骰子匣中的**一枚骰子**投掷，点数 = 移动步数。

- 玩家选择**顺时针或逆时针**移动
- 落点格子**必须触发**，无法跳过
- 环形棋盘可套圈（高点数绕多圈），每多走一圈「诅咒槽」+1
- 移动后该骰子不消耗，**归还骰子匣**，下次移动可继续使用

### 6.7 骰子用途二：嵌入赋能（战斗骰子）

参见 [第 9 章 战斗系统](#9-战斗系统战前嵌入--战中直伤) 完整说明。简述：

- 战斗开始前，从骰子匣选择骰子**嵌入装备骰槽**
- 嵌入消耗信仰值 = **骰子面数 ÷ 2**（D6=3，D12=6，D20=10）
- 同一 Run 内重复嵌入同一槽位**半价**
- 嵌入后**整场战斗锁定**，不可更换
- 武器嵌入后获得骰子伤害公式（如 `1d6+5`）；防具嵌入后获得护盾/护甲公式
- 战斗不再每回合分配骰子，改为**战前构筑 + 战中直伤**（详见第 9 章）

### 6.8 骰子获取途径

| 途径 | 说明 | 品质范围 |
|------|------|----------|
| **战斗掉落** | 普通战低概率掉骰子，精英战概率提升 | 白/蓝 |
| **BOSS 掉落** | 每层 BOSS 必掉 1 枚，品质随层数 | 蓝/紫/金 |
| **商店购买** | 随机刷新 1-2 枚骰子，可金币或信仰购买 | 白/蓝/紫 |
| **宝箱（宝藏格）** | 概率出骰子，层数越高品质越好 | 白/蓝/紫 |
| **骰子合成** | 2 枚同面数骰子 → 1 枚下一级骰子（2D6→D8→D10→D12→D20）；同品质 2 枚可重铸词条 | 视材料 |

**骰子合成（村庄铁匠铺/次元研究所）**：
```
面数进化：2×D6 → 1×D8 → 2×D8 → 1×D10 → 2×D10 → 1×D12 → 2×D12 → 1×D20
词条重铸：2×同品质骰子 → 1×同品质骰子（保留面数，刷新全部词条）
```

## 9. 战斗系统（骰子嵌入制）

> **核心设计理念**：战斗不再每回合分配骰子，而是**战前将骰子嵌入装备骰槽**，骰子面数直接决定装备的伤害公式。战斗中每回合选择攻击/防御/技能，伤害由已嵌入的骰子公式自动计算。策略重心从「每回合解谜」转移到「战前构筑」——选哪颗骰子嵌入哪件装备，是每场战斗的核心决策。

### 9.1 战斗概述

战斗采用**骰子嵌入制**（Dice Embedding）：

- **战前**：玩家从骰子匣选择骰子，消耗信仰值嵌入装备骰槽，激活装备的骰子伤害公式
- **战中**：每回合直接选择行动（普攻/技能/道具/防御），伤害由嵌入骰子自动驱动
- **嵌入锁定**：战斗中不可更换骰子，战前决策即是本场战斗的构筑
- **敌人意图系统**：保留，敌人在玩家行动前预告本回合行动

**核心决策点（战前）**：
- 这颗 D12 嵌入武器换高伤害，还是嵌入胸甲换高护甲？
- D100 嵌入哪个槽位能最大化传奇词条收益？
- 信仰值有限，这场战斗嵌入几颗骰子？

**核心决策点（战中）**：
- 看到敌人蓄力 → 本回合用防御（嵌入防具的骰子提供护盾）
- 看到敌人虚弱 → 全力攻击（嵌入武器的骰子释放高伤害）
- 血量危险 → 使用道具或技能

### 9.2 战前准备：骰子嵌入

每场战斗开始前，进入**嵌入阶段**：

```
┌─────────────────────────────────────────────────────────┐
│                    嵌入阶段                              │
│                                                         │
│  ① 打开骰子匣，查看所有可用骰子                          │
│      ↓                                                  │
│  ② 将骰子拖入装备的骰槽（武器/防具/副手等）             │
│      ↓                                                  │
│  ③ 消耗信仰值 = 骰子面数 ÷ 2（向下取整）                │
│      ↓                                                  │
│  ④ 骰子嵌入 → 装备激活骰子伤害公式 + 词条生效           │
│      ↓                                                  │
│  ⑤ 确认 → 进入战斗                                      │
└─────────────────────────────────────────────────────────┘
```

**嵌入规则**：

| 规则 | 说明 |
|------|------|
| 嵌入消耗 | 信仰值 = 骰子面数 ÷ 2（D6=3, D12=6, D20=10） |
| 重复嵌入 | 同一 Run 内同一槽位再次嵌入，信仰消耗减半 |
| 战斗中锁定 | 嵌入后整场战斗不可更换 |
| 可空槽 | 装备槽位可以留空（使用装备基础属性，无骰子公式） |
| 一颗骰子一槽 | 每颗骰子只能嵌入一个槽位 |

**嵌入效果**：
- 武器嵌入 → 获得骰子伤害公式（如嵌入 D6 → `1d6+武器基础`）
- 防具嵌入 → 获得护盾/护甲公式（如嵌入 D8 → `1d8+防具基础` 护盾）
- 戒指等饰品 → 词条加成翻倍

### 9.3 回合流程（简化）

```
┌─────────────────────────────────────────────────────────┐
│                    一回合战斗流程                         │
│                                                         │
│  ① 敌意阶段：敌人显示本回合意图（攻击/防御/技能）       │
│      ↓                                                  │
│  ② 行动阶段：玩家选择行动（普攻/技能/道具/防御/撤退）   │
│      ↓                                                  │
│  ③ 结算阶段：按速度排序结算所有行动                      │
│      ↓                                                  │
│  ④ 回合结束：清除临时状态，进入下一回合                  │
│      ↓                                                  │
│  重复直到一方全灭或逃跑                                  │
└─────────────────────────────────────────────────────────┘
```

**与旧版差异**：
- ❌ 删除：每回合投战斗骰子
- ❌ 删除：骰子分配/拖拽到 11 个槽位
- ❌ 删除：对子/顺子/总和≥N 等复杂条件
- ✅ 保留：敌人意图预告（让玩家有应对窗口）
- ✅ 保留：骰子伤害公式（XdY+Z 格式，由嵌入骰子决定）
- ✅ 新增：战前嵌入阶段（策略重心）

**玩家可用行动**：

| 行动 | 说明 | 条件 |
|------|------|------|
| 普攻 | 使用主手武器攻击，伤害由嵌入骰子公式计算 | 需主手武器嵌入骰子 |
| 副手攻击 | 使用副手武器攻击 | 需副手武器嵌入骰子 |
| 防御 | 获得护盾/护甲，数值由嵌入防具的骰子公式计算 | 需防具嵌入骰子 |
| 技能 | 使用套装技能或装备技能 | 部分需要骰子嵌入 |
| 道具 | 使用消耗品（不消耗骰子，直接使用） | 背包中持有 |

### 9.4 骰子伤害公式（继承 Elona 式）

伤害公式沿用 **XdY+Z(C)[±R]** 格式，其中 X, Y 由**嵌入的骰子**决定：

```
伤害 = 投 Y 面骰 X 次求和 + Z，暴击率 C%，最终波动 ±R
```

| 符号 | 含义 | 由什么决定 |
|------|------|-----------|
| **X** | 骰子数量 | 武器基础（1）+ 信念加成 |
| **Y** | 骰子面数 | **嵌入骰子的面数**（D6→6, D12→12, D100→100） |
| **+Z** | 固定加值 | 武器基础加值 + 属性加成 |
| **(C)** | 暴击率 | 武器基础暴击 + 敏捷加成 + 骰子词条 |
| **[±R]** | 最终波动 | 武器品级决定 |

#### 武器嵌入示例

| 武器 | 基础属性 | 嵌入骰子 | 激活伤害公式 | 说明 |
|------|----------|----------|-------------|------|
| 长剑 | 加值+3, 暴击5%, 波动±2 | D6（白） | `1d6+3(5)[-2,2]` | 早期标准配置 |
| 长剑 | 加值+3, 暴击5%, 波动±2 | D12（紫） | `1d12+3(5)[-2,2]` | 中期大幅升级 |
| 骑士重锤 | 加值+5, 暴击8%, 波动±3, X=2 | D20（金） | `2d20+5(8)[-3,3]` | 后期终极配置 |
| 修士法器 | 加值+8, 暴击3%, 波动±1 | D100（金） | `1d100+8(3)[-1,1]` | 传说骰，极端波动 |

#### 属性对公式的影响

| 属性 | 影响 |
|------|------|
| **力量（STR）** | 物理武器：每 5 点力量，固定加值 Z+1 |
| **智力（INT）** | 法术/信仰技能：每 5 点智力，伤害骰面数 Y 等效+1 |
| **敏捷（AGI）** | 每 8 点敏捷，行动优先级+1；每 10 点敏捷，暴击率 C+1% |
| **信念（FAI）** | 信仰技能：每 5 点信念，骰子数量 X+1（上限+2） |

### 9.5 敌人意图系统

在玩家选择行动前，敌人**预告本回合行动**，让玩家可以针对性应对：

| 意图图标 | 意图类型 | 说明 | 玩家应对 |
|---------|---------|------|----------|
| ⚔️ | **攻击** | 对玩家造成 X 点伤害 | 选择防御叠护盾 |
| 🛡️ | **防御** | 获得护盾/护甲 | 使用道具或等待 |
| ✨ | **技能** | 特殊技能（治疗/召唤/诅咒） | 集中攻击打断 |
| 🔥 | **强力攻击** | 高伤害，需 1 回合蓄力 | 必须防御或击杀 |

### 9.6 战斗特色：棋盘骰子联动

部分技能与**本局棋盘骰子点数**挂钩：

- 「神判之刃」：伤害公式中 Z 值 = 本层最后一次棋盘骰子点数
- 「幸运祈祷」：若本层棋盘骰子点数和 > 15，本场战斗首回合伤害+20%
- 「命运翻转」：将本层已用棋盘骰子点数总和转化为首回合额外伤害

### 9.7 状态效果

| 状态 | 效果 | 持续 |
|------|------|------|
| 圣洁 | 每回合结束时恢复 2HP，可叠加 | 永久 |
| 祝福 | 本场战斗骰子公式 Y 面数等效+2 | 3 回合 |
| 诅咒印记 | 每回合结束时受到诅咒层数×1 点伤害 | 5 回合 |
| 异端之火 | 每回合结束时造成灼烧伤害 | 3 回合 |

| 神怒 | 所有伤害+50%，但每回合-10HP | 3 回合 |

## 10. 装备与 Build 系统

> **核心设计理念**：游戏没有固定职业。玩家通过收集和穿戴装备，自由组合出战斗风格。装备套装触发专属技能，形成类职业的体验，但完全由玩家自主选择。

### 10.1 属性系统

玩家有以下核心属性，均可通过装备、遗物、升级提升：

| 属性            | 说明                                                 |
| --------------- | ---------------------------------------------------- |
| **力量（STR）** | 增加物理攻击伤害；**提升负重上限**（每点+2kg）       |
| **敏捷（AGI）** | 每 8 点增加行动优先级+1；增加闪避率；每 10 点暴击率 C+1% |
| **智力（INT）** | 增加法术/信仰技能伤害；增加技能效果数值          |
| **体质（CON）** | 增加最大 HP；提升护甲值上限                          |
| **信念（FAI）** | 提高信仰值上限；信仰获取效率+（与信仰系统深度联动）  |

### 10.2 装备槽位

```
┌────────────────────────────────────────┐
│  头盔   │  胸甲   │  护腕   │  腿甲    │
│  武器   │  副手   │  戒指×2 │  项链    │
│  披风   │  腰带   │  靴子   │          │
└────────────────────────────────────────┘
```

- 共 **11 个槽位**（戒指占 2 格）
- 每件装备有**重量值（kg）**
- 玩家**负重上限 = 基础 10kg + 力量 ×2kg**
- 每件装备有一个**骰槽**（Dice Socket），可嵌入一枚骰子

### 10.3 装备基础属性与骰子赋能

装备在**嵌入骰子之前**仅提供基础属性，嵌入骰子后释放其真正力量：

```
装备 = 基础属性（固定） + 骰子赋能（由嵌入骰子决定）

┌──────────────────────────────────────────┐
│  长剑（基础）                              │
│  基础伤害：5-8                             │
│  基础暴击：5%                              │
│  重量：3kg                                 │
│                                            │
│  骰槽：[ 空 ]  ← 嵌入骰子激活骰子公式      │
│  ─────────────────────────────             │
│  嵌入 D6  → 伤害公式 1d6+3(5)[-2,2]        │
│  嵌入 D12 → 伤害公式 1d12+3(5)[-2,2]       │
│  嵌入 D20 → 伤害公式 1d20+3(5)[-2,2]       │
└──────────────────────────────────────────┘
```

**装备部件与骰子赋能效果**：

| 装备部位 | 基础属性 | 嵌入骰子后 |
|----------|----------|-----------|
| 武器 | 固定伤害范围 + 基础暴击率 | 激活骰子伤害公式（XdY+Z(C)[±R]） |
| 防具（胸甲/头盔/腿甲等） | 固定护甲值 | 激活护盾公式 + 骰子通用词条 |
| 副手（盾牌） | 固定格挡值 | 护盾值 = 骰子面数 × 格挡系数 |
| 饰品（戒指/项链） | 属性 +1~2 | 骰子词条效果翻倍 |

**嵌入规则**：
- 嵌入消耗信仰值 = 骰子面数 ÷ 2（向下取整）
- 每件装备同一时间只能嵌入 1 颗骰子
- 战斗中锁定，不可更换
- 同一 Run 内同一槽位再次嵌入，信仰消耗减半
- 装备可以留空（骰槽为空），仅享受基础属性

### 10.4 装备套装与专属技能

穿戴同一套装的**足够件数**后，自动解锁该套装的专属被动/主动技能：

#### 🛡️ 骑士套装（Knight's Set）

> 沉重铁甲，神圣加持，庄严而不可撼动

**散件基础属性与骰子赋能**：

| 散件 | 基础属性 | 嵌入骰子后 |

## 11. 负重与背包系统

> 重量是战略资源，不只是限制。

### 11.1 负重机制

```
负重上限 = 10kg（基础）+ 力量属性 × 2kg
当前负重 = 所有已装备物品重量之和 + 背包中携带物品重量
```

**负重状态**：

| 状态        | 条件             | 效果                                       |
| ----------- | ---------------- | ------------------------------------------ |
| ✅ 轻装     | 负重 ≤ 上限 ×50% | 闪避率+10%，骰子结果+1（每层 1 次）        |
| ⚖️ 正常     | 50%~80%          | 无额外效果                                 |
| ⚠️ 超重     | 80%~100%         | 移速动画变慢（演出效果），战斗先手概率-20% |
| ❌ 严重超重 | >100%            | 每踩 1 格受到 2 点疲劳伤害，战斗中伤害 -30% |

### 11.2 背包系统

- 玩家有一个**普通背包**：初始容量 **6 格**，每格 1kg 负重计入
- 背包中的物品（装备/遗物/消耗品）**不在身上时不触发被动效果**
- 可在塔中商店购买背包扩容（+2 格，有上限）

### 11.3 次元空间（Dimensional Pouch）

**次元空间**是一种特殊的空间扩展法术/遗物，提供**不计入负重的额外储存格**：

| 名称     | 获得方式          | 容量           | 特性                          |
| -------- | ----------------- | -------------- | ----------------------------- |
| 次元小袋 | 商店购买/宝藏获得 | +4 格，0 重量  | 基础次元容器                  |
| 异端法阵 | 异端格子事件      | +8 格，0 重量  | 代价：进入时-2 信仰           |
| 神圣容器 | 圣化格奖励        | +6 格，0 重量  | 存放的装备获得+1 圣洁层数     |
| 虚空裂缝 | BOSS 掉落（稀有） | +16 格，0 重量 | 随机：取出物品时有 5%概率消失 |

**次元空间限制**：

- 次元空间中的物品**不影响负重**，但**取用有成本**（消耗 2 点信仰值或特定条件）
- 某些格子事件可能导致次元空间**不稳定**，暂时无法访问
- 村庄升级后可在「仓库」永久存放部分物品（下一次 Run 可携带出发）

### 11.4 管理策略示例

```
场景：玩家背包满了，发现了一件强力的骑士胸甲
选项A：装备它，卸下现有胸甲丢弃
选项B：放入次元空间（消耗 2 点信仰值）
选项C：放入背包，但超重了——接下来几格会受到疲劳伤害
选项D：回村时存入仓库（需先结束本层）
```

## 12. 信仰系统

> 信仰是本游戏最核心的差异化机制——它在塔内是战斗资源，在村庄中是建设货币，两条线共享同一个"信仰池"，形成取舍张力。

### 12.1 信仰值（Faith Points）

- **塔内信仰**：Run 内使用，初始 **5 点**（满值 20 点，可通过村庄升级提高上限）
- **村庄信仰**：每次 Run 结算后，玩家选择将部分塔内信仰贡献给村庄（不可撤回）
- **二者权衡**：留在塔内的信仰提升当局胜率；贡献给村庄的信仰推动长期养成

### 12.2 塔内获得与消耗

**获得方式**：

- 踩到祈祷所 +2~4
- 做出"信仰选项"的事件选择 +1~3
- 击败亵渎型怪物 +1
- 轻装状态下踩到圣化格 +3
- 某些装备/遗物被动获取

**消耗方式**：

- 使用信仰骰（指定点数）：-4
- 激活套装专属神力技能：-3~8
- 在商店购买圣物：-3~8
- 选择"亵渎选项"：-2~4

### 12.3 信仰标签系统

玩家积累信仰选择，会获得**标签**，影响全局效果：

| 标签         | 获得条件              | 效果                              |
| ------------ | --------------------- | --------------------------------- |
| **圣战者**   | 信仰值 5 次达到最大值 | 攻击亵渎型敌人+50%伤害            |
| **异端者**   | 3 次选择亵渎选项      | 解锁异端道具池，信仰上限-5            |
| **殉道者**   | HP 低于 20%时触发战斗 | 低血量时攻击力+2，但不能回血      |
| **怀疑者**   | 5 次拒绝神明指引      | 所有随机事件结果可重投一次        |
| **虔信者**   | 全程信仰值不低于 10   | 每层开始额外获得一件圣洁道具          |
| **慷慨村长** | 累计贡献信仰值 ≥ 50   | 村庄加成反哺塔内：每层起始+1 信仰 |

## 13. 村庄养成系统

> 玩家是村长，塔是副业，村庄是家。每次冒险都是为了让村子活下去，变得更好。

### 13.1 系统概览

村庄养成是**跨 Run 的永久进度系统**，与爬塔形成**资源循环**：

```
爬塔 → 获取资源（金币/材料/信仰） → 贡献给村庄
                                       ↓
                              村庄升级 → 解锁新建筑/NPC
                                       ↓
                              新建筑/NPC → 提供爬塔增益
                                       ↓
                              更强地爬塔 → 获取更多资源
```

### 13.2 村庄等级（定居点进化）

| 等级 | 名称     | 升级条件                   | 解锁内容                       |
| ---- | -------- | -------------------------- | ------------------------------ |
| 1    | **村落** | 初始状态                   | 基础设施（家、祈祷场、出发点） |
| 2    | **村庄** | 贡献资源 ×50 + 信仰 ×20    | 铁匠铺、市场、第 1 个建筑槽    |
| 3    | **镇子** | 贡献资源 ×200 + 信仰 ×80   | 城墙、教堂、招募高级村民       |
| 4    | **城市** | 贡献资源 ×600 + 信仰 ×200  | 大学、骑士团总部、商会         |
| 5    | **圣城** | 特殊条件（完成真结局分支） | 神殿、永久爬塔增益最大化       |

### 13.3 信仰值在村庄中的功能

信仰值是村庄的**第二货币**，专用于以下功能：

| 功能           | 消耗信仰     | 效果                                    |
| -------------- | ------------ | --------------------------------------- |
| **招募村民**   | 5~20 信仰/人 | 村民执行各类任务，人口增加税收          |
| **升级定居点** | 参见上表     | 推进村庄等级                            |

| **赦免罪人**   | 10 信仰      | 将因事件失去的村民找回                  |

### 13.4 税收系统

- **人口**决定税收基础：每个村民每次 Run 结束贡献 1~3 金币（取决于职业）

- **建筑**提供税收加成（如市场+20%，城墙+5%安全保障）
- 玩家 Run 开始前，从村庄领取**出发补给**（金币+可选消耗品）

## 14. 世界地图与塔系统

> **核心设计理念**：高塔不再唯一，而是林立在世界各地。村庄在地图中央，玩家从地图室出发，选择一座塔进行探索。塔有身份、等级、层级三个独立维度，每次进入随机生成，形成类副本的探索体验。

### 14.1 世界概述

```
                     [异端要塞 Lv.5]
                           │
              [瘟疫地窖 Lv.3]──┼──[钟楼残骸 Lv.4]
                           │
          [圣堂废墟 Lv.1]──┼──[沉默修道院 Lv.2]──[? 未探索]
                           │
                     ★ 村庄（中心）★
                           │
              [? 未探索]──┼──[地窖试炼 Lv.1]
                           │
                     [? 未探索]
```

- **村庄始终在世界中心**
- 塔散布在村庄四方，通过道路连接
- 已发现的塔永久可见，显示名称、等级、状态
- 未探索区域以**迷雾**遮盖，派遣斥候可逐步揭示

### 14.2 地图室（村庄建筑）

地图室是玩家在世界中选择方向的战略界面：

```
┌──────────────────────────────────────────────────────┐
│                 世界地图（地图室）                      │
│                                                      │
│              [瘟疫地窖 Lv.3 ⚔️]                       │
│                    │                                 │
│  [圣堂废墟 Lv.1 ✓]─┼──[异端要塞 Lv.5 🔒]             │
│                    │                                 │
│              ★ 村庄 ★──[? 迷雾]                       │
│                                                      │
│  [出发补给] [选择塔] [派遣斥候] [返回村庄]            │
└──────────────────────────────────────────────────────┘
```

**地图室操作**：
- **查看塔**：点击已发现的塔，显示名称/等级/层级/特殊规则/推荐战力
- **派遣斥候**：消耗 12 信仰，斥候揭示一个方向的迷雾，可能发现新塔
- **选择出发**：选塔后进入出发准备（嵌入骰子、装备、补给），消耗旅途补给进入

| 瘟疫地窖 | 地下墓地 | 疫病鼠群、尸鬼 | 每 3 回合全体中毒 | 肿胀畸变体 |
| 异端要塞 | 军事堡垒 | 异端骑士、火焰法师 | 所有敌人护甲翻倍 | 异端大团长 |
| 沉默修道院 | 山中隐修 | 沉默僧侣、光之圣灵 | 无法使用信仰技能 | 圣母像·暗面 |
| 钟楼残骸 | 崩塌高塔 | 石像鬼、钟楼守卫 | 每层时间限制（20 回合） | 钟楼守护者 |

**塔身份定义**：
- 一座塔 = 一个主题 + 一个敌人池 + 1-2 条特殊规则 + 固定 BOSS
- 塔一旦发现就是**永久可进入的副本**
- 同一 Run 只进入一座塔

### 14.4 塔的等级与层级（正交维度）

塔的**等级**和**层级**是两个独立维度：

#### 塔等级（难度轴，固定）

| 塔等级 | 怪物属性倍率 | 掉落最低品质 | D100 出现率 | 推荐 |
|--------|-------------|-------------|------------|------|
| Lv.1 | ×0.8 | 白 | 0% | 新手 |
| Lv.2 | ×1.0 | 白 | 0% | 标准 |
| Lv.3 | ×1.3 | 蓝 | 0.1% | 困难 |
| Lv.4 | ×1.6 | 蓝+ | 0.5% | 噩梦 |
| Lv.5 | ×2.0 | 紫 | 2% | 地狱 |

- 塔等级是**固定的**（异端要塞永远是 Lv.5）
- 高等级塔需要先通关相邻低等级塔来**解锁进入权限**

#### 塔层级（深度，每次进入随机）

| 层级数 | 说明 | 出现概率 |
|--------|------|----------|
| 3 层 | 浅塔，速刷型 | 30% |
| 4-6 层 | 标准深度 | 50% |
| 7-9 层 | 深塔，远征型 | 15% |
| 10 层 | 极深塔，终局 | 5% |

- 层级数**每次进入塔时随机生成**，不受塔等级影响
- 层级多 ≠ 更难（低等级塔 10 层也是低级怪），但代表**探索时间长**
- 每层格子数 = 24 + 层级 × 2（24~44 格），每层递增 2 格

### 14.5 塔内随机生成（每次进入）

| 要素 | 生成规则 |
|------|----------|
| 层级数 | 随机 3~10 层 |
| 每层格子数 | 24 + 层数 × 2，随机微调 ±2 |
| 格子分布 | 按模板比例随机排列（战斗 40%/事件 20%/商店 10%/祈祷所 10%/宝藏 8%/陷阱 7%/BOSS 固定） |
| BOSS 格 | 每层末位固定 1 个 BOSS 格 |
| 稀有事件 | 概率 = 5% + 塔等级×3%（Lv.1=8%, Lv.5=20%） |

**生成锚点**：塔 ID（种子锚） + Run 种子 → 确定性随机（同种子同塔 = 相同布局）

### 14.6 塔的发现与解锁

| 方式 | 条件 | 说明 |
|------|------|------|
| 初始赠送 | 开局 | 村庄周围 2 座 Lv.1 塔 |
| 斥候探索 | 招募斥候 + 消耗 12 信仰 | 揭示一个方向，发现 1-2 座新塔 |
| 通关解锁 | 通关某座塔 | 该塔相邻的未发现塔自动揭示 |
| 特殊事件 | 村庄事件 / 传教士 | 低概率带回远方塔的情报 |

**塔等级递进解锁**：
- 通关任意 Lv.N 塔 → 解锁相邻 Lv.(N+1) 塔的进入权限
- 初始可进入 Lv.1 塔
- 未解锁的高等级塔在地图上显示为 🔒

### 14.7 旅途消耗

每次出发探索一座塔需要消耗**旅途补给**：

| 塔等级 | 旅途补给消耗 |
|--------|-------------|
| Lv.1-2 | 1 份补给 |
| Lv.3 | 2 份补给 |
| Lv.4 | 3 份补给 |
| Lv.5 | 5 份补给 |

- **补给来源**：村庄税收购买（1 金币 = 1 份）、村民生产、Run 结算奖励
- 补给不足时无法进入高等级塔（避免一上来就送死）
- 轻装效果：补给消耗减 1（最低 1）

### 14.8 战斗奖励（三选一系统）

战斗胜利后，系统从奖励池中随机生成 **3 个选项**，玩家**必须选择 1 个**。奖励品质受**塔等级**影响：

#### 奖励池类型

| 类型 | 图标 | 说明 | 普通战概率 | 精英战概率 |
|------|------|------|-----------|-----------|
| **道具** | 🧪 | 消耗品（回复药、投掷物、增益卷轴等） | 30% | 15% |
| **装备** | 🗡️ | 随机品质的装备件 | 25% | 40% |
| **金币** | 💰 | 一笔金币奖励（10~50，随塔等级递增） | 25% | 15% |
| **骰子** | 🎲 | 随机品质骰子（新增奖励类型） | 10% | 15% |
| **状态恢复** | 💚 | HP 恢复 / 信仰恢复 / 诅咒清除等 | 10% | 5% |
| **遗物** | 📿 | 被动效果遗物（仅精英战/BOSS战） | 0% | 10% |

#### 品质权重（由塔等级决定）

| 塔等级 | 普通 | 精良 | 稀有 | 传说 |
|--------|------|------|------|------|
| Lv.1 | 70% | 25% | 5% | 0% |
| Lv.2 | 50% | 35% | 14% | 1% |
| Lv.3 | 30% | 35% | 30% | 5% |
| Lv.4 | 15% | 35% | 35% | 15% |
| Lv.5 | 5% | 25% | 40% | 30% |

#### 状态恢复细分

| 恢复类型 | 效果 | 出现条件 |
|----------|------|----------|
| 生命恢复 | 恢复 20%~40% 最大HP | 当前HP < 80% |
| 信仰恢复 | 恢复 3~5 点信仰值 | 当前信仰值 < 上限50% |
| 诅咒清除 | 降低 2~4 层诅咒 | 当前诅咒 ≥ 3 |
| 负重减轻 | 本层剩余移动中负重减半 | 当前负重 ≥ 80% |
| 全面休整 | 恢复 10%HP + 2信仰 + 1诅咒清除 | 随机低概率 |

### 14.9 层间结算

每层清完后进入「层间神坛」：

- 从 2 件装备/遗物/骰子中选 1 个
- 可选择「净化诅咒」（-5 金）或「强化一件装备」（-10 金）
- **村庄贡献选项**：将本层获得的部分资源捐给村庄（即时转化为村庄进度）

### 14.10 通关与返回

- 击败塔顶 BOSS → 塔通关 → **获得塔通关奖励**（大量金币 + 信仰 + 稀有骰子）
- 可选择「返回村庄」或「在塔内继续绕圈刷资源」（每多一圈诅咒 +2）
- 死亡 → 失去本 Run 塔内未贡献的资源，保留已贡献的村庄进度
- 同一 Run 可连续探索多座塔（需返回村庄 → 重新出发），但旅途补给有限

## 15. 遗物与战斗奖励系统

### 15.1 道具分类（消耗品）

战斗奖励中的「道具」为一次性消耗品，使用后消失：

| 类别 | 说明 | 示例 |
|------|------|------|
| **回复类** | 恢复HP/信仰值 | 治愈药水（+20HP）、祈祷珠（+3信仰） |
| **攻击类** | 直接对敌造成伤害 | 圣水瓶（对亵渎敌人30伤害）、火油壶（群体10伤害） |
| **增益类** | 临时增强玩家属性 | 力量卷轴（本场战斗STR+5）、疾风卷轴（骰子池+1颗，3回合） |

**道具限制**：
- 背包中最多携带 **8 个道具**（不受次元空间影响）
- 同类道具可叠加，上限 3 个
- 战斗中使用道具消耗 1 颗任意骰子（将骰子投入道具槽位激活）

### 15.2 遗物系统

| 骰盘护符     | 每局游戏可重投骰子 1 次                            |
| 忏悔室钥匙   | 进入商店时额外获得一个免费选项                     |

| 村长印章     | 携带时，每次 Run 结束从村庄额外领取 10 金币        |

**遗物限制**：

- 遗物无法丢弃，但可在商店「净化」移除（获得少量金币）

## 16. BOSS 设计

### 16.1 BOSS 机制通则

- 每座塔的**顶层固定一个 BOSS**，塔身份决定 BOSS 池（每座塔 1-2 个候选 BOSS，随机抽 1）
- BOSS 有专属像素立绘和战斗主题曲
- BOSS 有**阶段变化**：通常 2 个阶段
- BOSS 会根据**玩家装备套装**和**信仰值**调整策略：
  - 骑士套装满激活 → BOSS 强化对护甲的穿透
  - 高信仰 → BOSS 使用"亵渎攻击"克制圣洁
  - 低信仰 → BOSS 使用"秩序惩罚"攻击异端

### 16.2 BOSS 与塔对应

| BOSS 名 | 所属塔（示例）| 主题 | 塔等级 | 特殊机制 |
|---------|-------------|------|--------|----------|
| 腐败审判官 | 圣堂废墟 | 堕落的秩序 | Lv.1-2 | 每回合对玩家施加 1 层诅咒印记 |
| 肿胀畸变体 | 瘟疫地窖 | 疫病扩散 | Lv.1-2 | 每损失 20%HP 召唤一波鼠群 |
| 圣母像·暗面 | 沉默修道院 | 信仰的扭曲 | Lv.2-3 | 模仿玩家上一次使用的技能 |
| 钟楼守护者 | 钟楼残骸 | 时间的重量 | Lv.3-4 | 每 5 回合加速诅咒 2 层 |
| 异端大团长 | 异端要塞 | 异端审判 | Lv.4-5 | 对嵌入骰子的槽位造成额外伤害 |
| 大祭司遗体 | 圣堂废墟(深) | 教会的牺牲 | Lv.3-5 | 每受到一次攻击恢复少量 HP |
| 神明本体·「真理」| 终局塔 | 终极审判 | Lv.5 | 根据信仰标签+村庄等级改变形态 |

### 16.3 BOSS 掉落

## 17. 肉鸽要素

### 17.1 每局随机内容

- **塔内层级数**：每次进入随机（3~10 层）
- **棋盘格子排布**：完全随机
- **怪物组合**：随机（含词缀系统），由塔身份限定族群范围
- **BOSS 二选一**：同一座塔的 BOSS 从候选池随机抽 1
- **道具/装备/骰子奖励**：随机（受塔等级影响品质）
- **遗物随机**：共约 60 个，每局约见到 15~20 个
- **事件随机**：共约 40 个，每局约触发 10~15 个
- **商店货架随机**

### 17.2 永久解锁（Meta 进度）

| 解锁条件 | 解锁内容 |
|----------|----------|
| 首次通关任意塔 | 地图室解锁；可派遣斥候探索 |
| 通关 Lv.3 塔 | 困难模式（高等级塔诅咒加速） |
| 村庄升至镇子 | 解锁稀有装备池 |
| 发现 10 座塔 | 特殊遗物「旅人之星」永久解锁（旅途补给消耗-1） |
| 获得全部信仰标签 | 真结局分支解锁 |
| 完成 10 次殉道者通关 | 特殊遗物「殉道者圣冠」永久解锁 |
| 村庄升至圣城 | 最终结局解锁（终极塔「神之眼」出现），特殊开场叙事 |

### 17.3 诅咒等级（Run 内难度调节）

| 诅咒层数 | 效果                         |
| -------- | ---------------------------- |
| 0-2      | 无附加效果                   |
| 3-5      | 怪物+10% HP                  |
| 6-8      | 商店价格 ×1.5                |
| 9-11     | 每回合棋盘随机移动 1 格      |
| 12+      | 每步移动额外触发随机负面事件 |

## 19. 音效与音乐方向

### 19.1 音乐风格

- **主题**：中世纪圣歌 + 低保真电子节拍（Lo-fi Gregorian）
- **村庄 BGM**：温暖羽管键琴 + 自然音效（风声、鸟鸣），祥和感
- **棋盘探索 BGM**：轻缓管风琴 + 轻鼓点
- **战斗 BGM**：紧张弦乐 + 打击乐，精英战加入合唱片段
- **BOSS 战 BGM**：全力管风琴 + 黑暗合唱
- **神坛抉择 BGM**：空灵单音铃声，宁静而压迫
- **村庄危机 BGM**：不安的弦乐，低沉鼓点

### 19.2 音效设计

- 骰子投掷：骨头/石头碰撞声
- 棋子移动：石板摩擦声
- 格子激活：蜡烛点燃/门锁开启声
- 攻击音效：金属碰撞，法术有圣光/火焰质感
- 信仰获得：短促合唱音效（类似「Amen」片段）
- 获得诅咒：低沉嗡鸣声
- 套装激活：对应风格音效（骑士：号角；修士：钟声；异端：诡异低鸣）
- 负重超限：沉重喘息声
- 村庄升级：欢呼声 + 教堂钟声

## 附录 A：术语表

| 术语 | 说明 |
|------|------|
| Run | 一局完整游戏流程（从出发到死亡/通关） |
| Build | 本局装备套装+骰子嵌入+遗物+道具的搭配方案 |
| 骰子实体 | 可收集的核心道具，有品质、面数、词条三个维度 |
| 骰子面数 | D4 / D6 / D8 / D10 / D12 / D20 / D100，决定投掷范围和嵌入威力 |
| 词条 | 骰子附带的特殊效果，分投掷词条、嵌入词条、通用词条 |
| 骰子匣 | 骰子携带容器，初始 2 格，扩容是核心成长线 |
| 骰子嵌入 | 战前将骰子嵌入装备骰槽，消耗信仰值，激活骰子伤害公式 |
| 骰槽 | 每件装备上的骰子嵌入位，嵌入后战斗中锁定 |
| 棋盘骰子 | 从骰子匣选一枚骰子投掷用于移动，不消耗骰子 |
| 意图系统 | 敌人在玩家行动前预告本回合行动（攻击/防御/技能/蓄力） |
| 诅咒槽 | 绕圈惩罚机制，每多走一圈+1 诅咒 |
| 信仰值 | Faith Points，塔内嵌入资源 & 村庄建设货币 |
| 层间神坛 | 每层结束后的增益选择界面 |
| 负重 | 装备和背包总重量，超限有惩罚 |
| 次元空间 | 不计入负重的特殊储存容器 |
| 村庄等级 | 跨 Run 的永久养成进度 |
| 骰子合成 | 2 枚同面数骰子合成 1 枚升阶骰子（D6→D8→D10→D12→D20） |
| 多主题换皮 | 「骰境」系列特性，切换主题整体替换 UI/骰子材质/棋子/BGM |
