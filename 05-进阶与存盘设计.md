# 05 · 进阶等级（尖塔式）与单 Run 存盘

对齐纲要：**单局 5–8 小时**、**章节制**、**死亡整局重来**、**Meta 永久解锁**。

---

## 1. 进阶系统总规则 v0

- **解锁**：首次**任意难度**通关（击败第 99 层 Boss 或切片期等价终点）后开放「进阶 1」。
- **选用**：开局前选 **0–已解锁最高进阶**（可随时选低进阶刷图解锁，与尖塔一致）。
- **叠加**：每档进阶**追加**一条主效应 + 若干次要效应（下列表为 v0，可拆分实现）。

**禁止**：进阶中途升降；仅在下次开局时生效。

## 3. 自定义挑战（可选模式）

- **突变因子**（多选）：如「仅夜间害格」「骰子恒为 d4」「起始诅咒×2」。
- 与进阶**加算**或**独立排行榜**（避免平衡爆炸）；v0 建议 **独立**，不叠加进阶 10+。

---

## 4. 单 Run 存盘 / 断点规则

### 4.1 设计目标

- 支持 **5–8 小时**单次游玩的**现实中断**，不改变 Roguelike「死亡删 Run」原则。

### 4.2 自动存盘点（推荐全实装）

| 时机 | 行为 |
|------|------|
| 进入新一层（地图生成完毕） | 写入自动存盘槽 A |
| 击败 Boss、进入层间奖励界面后 | 写入自动存盘槽 A |
| 回到主菜单 / 应用退出 | 若有进行中 Run，提示「将保存当前进度」 |

### 4.3 手动存盘

- **暂停 → 保存并退出**：写入槽 A（与自动共用）或独立槽 B（二选一；尖塔式为单槽继续）。
- **限制**：战斗中不可写盘（或仅允许「回合间隙」），防止存档作弊刷随机。

### 4.4 死亡与放弃

- **死亡**：清除当前 Run 的继续档；保留 **Meta 进度**（解锁角色、图鉴、进阶阶数记录）。
- **放弃 Run**：同死亡，可选二次确认。

### 4.5 防作弊与 Seeds（工程可选）

- 存盘含 **随机种子 + Run 版本号**；版本不兼容则作废档提示。
- 进阶 / 自定义挑战在存盘头中记录，加载时校验。

### 4.6 SLA 对玩家的表述（文案）

> 「黑暗之塔会记录你当前的攀登进度。若你倒下，重来之路将从塔底开始；诸神不会宽恕，但你的所历所见将铭刻于命运之外（Meta）。」

---

## 5. 与 QA / 调时的建议

- 测 **0 阶 / 10 阶 / 20 阶** 三组通关率与中位时长，目标 **中位 5–8h（0–5 阶）**。
- 进阶 **≥12** 起必须监控「移动类消耗品」胜率贡献，避免唯道具解。
