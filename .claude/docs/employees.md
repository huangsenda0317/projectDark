# 三位员工介绍

本项目由三台电脑协作驱动，每台机器作为独立的 Agent 工作节点，各司其职。

---

## Kite（风筝）

| 属性 | 详情 |
|------|------|
| **角色** | 主控节点 / 协调者 |
| **机型** | MacBook (2020) |
| **操作系统** | macOS |
| **定位** | 日常开发主力的笔记本电脑，轻便灵活，适合随时随地编码 |
| **擅长** | 项目管理、代码审查、架构设计、文档编写 |
| **分工** | 负责任务分发、设计决策、代码评审和最终集成。大部分交互式开发在此机器上完成。 |

---

## Green（格林）

| 属性 | 详情 |
|------|------|
| **角色** | 后台构建节点 |
| **机型** | MacBook (2015) |
| **操作系统** | macOS |
| **定位** | 旧款但稳定的 MacBook，适合作为持续构建和测试的常驻机器 |
| **擅长** | 构建打包、CI/CD、自动化测试、长时间运行任务 |
| **分工** | 负责后台编译构建、自动化测试套件执行、资源烘焙等耗时任务。不进行交互式开发，专注于稳定输出。 |

---

## Max（麦克斯）

| 属性 | 详情 |
|------|------|
| **角色** | Windows 构建与测试节点 |
| **机型** | Windows 台式机 |
| **操作系统** | Windows |
| **定位** | 高性能 Windows 机器，负责跨平台验证和 Windows 原生构建 |
| **擅长** | Windows 构建、GPU 密集型任务、跨平台兼容性测试 |
| **分工** | 负责 Windows 平台的原生构建打包、跨平台兼容性验证、性能测试，以及利用 GPU 资源的任务（渲染测试、资源处理等）。 |

---

## 协作模式

```
用户 (huangsenda)
  │
  ├─ Kite (主控) ─── 任务分发、代码评审、架构决策
  │
  ├─ Green (后台) ── 构建打包、自动化测试、资源烘焙
  │
  └─ Max (Win) ──── Windows 构建、跨平台验证、性能测试
```

三台机器通过 Git 仓库同步代码，各自独立运行 Claude Code Agent，由用户统一调度。
