---

## 2. 第一章（1–33）· 涂血基石

### 2.1 氛围与机制倾斜

- **事件池**：教会伤害类（圣水、火刑余烬）中等权重；补给与魂屑略多，降低劝退。
- **迷雾**：以「路过仅类型」为主教学；act 末层再加强真随机害格。
- **视觉**：残墙、屠杀现场纪念品、断裂十字架；Boss 区改为临时审讯所。

### 2.2 Boss 线（建议每章 3 个大节点 + 章末）


| 层区间    | Boss 代号     | 身份          | 战点摘要                      |
| ------ | ----------- | ----------- | ------------------------- |
| ~11    | **执火祭司**    | 屠村带队司铎的低阶化身 | AOE 圣火；召唤火刑架障碍            |
| ~22    | **连环审判官**   | 追猎异端的武装修士   | 高单伤；玩家暴露值高时附加追击           |
| **33** | **苦修巨像·雏形** | 信仰堆砌的构造体    | 阶段1 物抗；阶段2 转向「自残式狂热」，攻击暴涨 |
| （备选层）  | 隐藏          | **告解灵**     | 以玩家曾选「伪神迹」次数强化，衔接第二章      |


### 2.3 小精英命名池

肃清者、烙铁仆从、赎罪链丧尸（低智能）。

---

## 3. 第二章（34–66）· 伪圣螺旋

### 3.1 氛围与机制倾斜

- **事件池**：抉择类（伪神迹、赌徒圣坛）↑；颠倒圣印、镜像池权重上升。
- **地图**：环形跑道易出现「假上楼格」——踩中进入支路 3 步净化才可回环。
- **叙事**：NPC 幻象、假先知；恶魔提示减少，改为环境叙事。

### 3.2 Boss 线


| 层区间    | Boss 代号   | 身份         | 战点摘要                 |
| ------ | --------- | ---------- | -------------------- |
| ~44    | **百舌合唱**  | 被缝合唱团的灵体集合 | 沉默 / 打断机制；层数叠音波      |
| ~55    | **玻璃殉道者** | 圣骸与玻璃刀片结合  | 反伤与「可见攻击预告」地板        |
| **66** | **信仰聚合体** | 民众祈愿的畸变集合  | 吸收玩家「信仰值」资源；需打散护轨再输出 |
| （隐藏）   | **伪圣子**   | 教堂偶像成精     | 若第一章选「信」多则强化；反之削弱    |


### 3.3 中点转折（约 50 层）

可选插入 **一次性剧情**：雕像流泪 / 天空倒转，解锁恶魔捷径事件（风险：多诅咒，跳过 5 层）。

---

## 4. 第三章（67–99）· 冠冕倒影

### 4.1 氛围与机制倾斜

- **事件池**：harm 与 special 权重大；resource 好物更稀有但更强力。
- **机制**：时砂流沙、肃清钟鸣高频；可配置「每章开始重置计数」防止无解。
- **美术**：逆重力雨、阶梯刺向天空；UI 可做轻微倾斜畸变（性能允许时）。

### 4.2 Boss 线


| 层区间    | Boss 代号       | 身份           | 战点摘要                     |
| ------ | ------------- | ------------ | ------------------------ |
| ~77    | **翼刃侍从**      | 堕距最近的天使武备    | 高速多段；逼迫用防御技能             |
| ~88    | **权杖宣讲者**     | 神言具象         | 「律令」封锁格 / 封锁技能槽（可与棋盘骰联动） |
| **99** | **冠冕残响**（真名锁） | 非完整神体 / 神之代理 | 多阶段；终阶段引用玩家周目内关键抉择台词     |


### 4.3 真结局门槛 v0（设计可选）

- **普通通关**：击败 99 层「冠冕残响」标准形态。
- **真结局**：需 **进阶 ≥10** 且持有「异端伪装」3 件套或完成恶魔契约线，终战多一段 **神格剥落** QTE / 机制战。

---

## 5. 与系统衔接清单

- **守关节奏**：每章末层（33 / 66 / 99）固定 **上楼 Boss**；中间 Boss 可置于「必经上楼格前」或独立圣所格。
- **音乐分层**：三章各一条主旋律变奏；Boss 共用「教会调式 → 微分音噪 → 寂静高频」。
- **事件表权重**：与 [01-事件表-v0.md](01-事件表-v0.md) 中 `w_act1/2/3` 对齐 act 段。
